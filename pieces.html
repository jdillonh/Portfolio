Synestaether

Synestaether is a peaceful 3D puzzle game about moving marbles around. Made over about 6 weeks for a game design studio class with 2 other students.

Technologies: Unity, Autodesk Maya, C#

Vynth

Vynth is a video synthesizer made with p5.js and WebGL. Intended to make the daunting technical difficulty of shader art more accesible to creative people. Patches created in the visual programming interface are compiled to GLSL fragment shaders.

Technologies: p5.js, WebGL, HTML, CSS

Bloink2

Bloink2 is a game where the player makes generative musical compositions by placing different physics objects in the world. Those objects interact to make rythmic and melodic material.

Technologies: JS, HTML, CSS
